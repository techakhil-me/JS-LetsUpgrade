Difference between var, let and const

    var
  1. The var variables belong to the global scope or local scope if they are declared inside a function:
  2. The global var variables are added to the global object as properties. The global object is window on the web browser and global on Node.js:
  3. However, the let variables are not added to the global object:
  4. 
function nodeSimplified(){
    var a =10;
    console.log(a);  // output 10
    if(true){
        var a=20;
        console.log(a); // output 20
    }
    console.log(a);  // output 20
}
    let
  1. The let variables are blocked scopes:
  2. However, the let variables are not added to the global object:
  3. If you redeclare a variable with the let keyword, you will get an error:
  4. 
function nodeSimplified(){
    let a =10;
    console.log(a);  // output 10
    if(true){
        let a=20;
        console.log(a); // output 20
    }
    console.log(a);  // output 10
}
    const
  1. The const statements are blocked scopes:
  2. Similarly, the const statements are not added to the global object:
  3. Const statement values can be assigned once and they cannot be reassigned:
  4. 
function nodeSimplified(){
    const MY_VARIABLE =10;
    console.log(MY_VARIABLE);  //output 10 
    }
