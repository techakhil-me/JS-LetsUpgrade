Types of data type in javascript

  1. log()
    2. Mainly used to log(print) the output to the console. We can put any type inside the log(), be it a string, array, object, boolean etc.
  2. error()
    3. Used to log error message to the console. Useful in testing of code. By default the error message will be highlighted with red color.
  3. warn()
    4. Used to log warning message to the console. By default the warning message will be highlighted with yellow color.
  4. clear()
    5. Used to clear the console.
  5. time()
    6. When you call console.timeEnd() with the same name, the browser will output the time, in milliseconds, that elapsed since the timer was started.
  6. timeEnd()
    7. Stops the specified timer and logs the elapsed time in seconds since it started.
  7. table()
    8. This method allows us to generate a table inside a console. The input must be an array or an object which will be shown as a table.
  8. count()
    9. This method is used to count the number that the function hit by this counting method.
  9. group()
    10. Creates a new inline group, indenting all following output by another level. To move back out a level, call groupEnd().
  10. groupEnd()
    11. Exits the current inline group.
