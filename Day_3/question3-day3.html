Types of console functions in javascript

  1. numbers: represents numeric values e.g. 100
  2. string: represents sequence of characters e.g. "hello"
  3. null: represents null i.e. no value at all
  4. undefined: represents undefined value
  5. boolean: represents boolean value either false or true
  6. NaN: The NaN property represents "Not-a-Number" value.
    This property indicates that a value is not a legal number.
